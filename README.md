# Smart Expense Tracker

A Full Stack Expense Tracking Application built using React, Flask, SQLite and JWT Authentication.

## Features

- User Authentication
- Add/Edit/Delete Transactions
- Budget Management
- Budget Alerts
- Expense Analytics
- Pie Chart Visualization
- Search Transactions
- Monthly Filter
- PDF Report Export
- Profile Page
- Dark Mode
- AI Spending Suggestions

## Tech Stack

Frontend:
- React
- Tailwind CSS
- Axios

Backend:
- Flask
- JWT Authentication
- SQLite

Charts:
- Recharts

PDF:
- jsPDF
