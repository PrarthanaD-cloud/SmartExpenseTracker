#  Smart Expense Tracker

A full-stack Expense Tracking Application built using React, Flask, SQLite, JWT Authentication, Tailwind CSS, and Recharts.

---

##  Features

### Authentication
- User Registration
- User Login
- JWT Authentication
- Protected Dashboard

### Transaction Management
- Add Transactions
- Delete Transactions
- Income Tracking
- Expense Tracking
- Category Based Transactions

### Budget Management
- Monthly Budget Setting
- Remaining Budget Calculation
- Budget Alerts

### Analytics
- Expense Analytics Dashboard
- Pie Chart Visualization
- Category Wise Expense Analysis

### Search & Filter
- Search Transactions
- Monthly Filter
- Category Search
- Description Search

### User Features
- Profile Page
- Dark Mode
- Light Mode

### Reports
- Export PDF Reports

### Smart Features
- AI Spending Suggestions
- Savings Insights
- Financial Health Monitoring

##  Installation

### Clone Repository

```bash
git clone https://github.com/PrarthanaD-cloud/SmartExpenseTracker.git
```

```bash
cd SmartExpenseTracker
```

---

## Backend Setup

```bash
cd backend
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Environment

```bash
venv\Scripts\activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Backend

```bash
python app.py
```

Backend runs on:

```text
http://127.0.0.1:5000
```

---

## Frontend Setup

Open New Terminal

```bash
cd frontend
```

Install Dependencies

```bash
npm install
```

Run Frontend

```bash
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```
